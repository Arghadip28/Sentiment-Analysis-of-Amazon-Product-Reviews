# Sentiment-Analysis-of-Amazon-Product-Reviews

## Project Overview
This project aims to analyze user sentiments from Amazon product reviews using state-of-the-art Natural Language Processing (NLP) and machine learning techniques. By classifying reviews as positive, negative, or neutral, it provides valuable insights into customer satisfaction and helps to uncover product strengths and areas for improvement.​

## Features
- Preprocessing of raw review data, including cleaning and normalization.

- Exploratory data analysis and visualization.

- Feature extraction using methods like Bag-of-Words, TF-IDF, and Word2Vec.​

- Model building using classifiers and/or neural networks (e.g., LSTM).

- Accuracy evaluation and sentiment prediction on unseen data.

- Export of results and insights.

## Installation and Setup
### Clone the repository
```bash=
git clone https://github.com/Arghadip28/Sentiment-Analysis-of-Amazon-Product-Reviews.git
cd Sentiment-Analysis-of-Amazon-Product-Reviews
```

### Install dependencies
```bash=
pip install -r requirements.txt
```

### Run the Jupyter Notebooks
Open and execute the notebooks for exploratory data analysis, preprocessing, and model training.

## Usage
- Run the notebooks or Python scripts for preprocessing and training.

- Predict sentiment by supplying new review texts.

- Explore data visualizations and model evaluation outputs.

- Modify or extend the code for additional NLP experiments.
